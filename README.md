##项目名称和简要描述

  淘淘乐是一个面向大学生的二手物品交易平台，项目旨在为学生提供一个安全、便捷的闲置物品交易平台。

##功能列表
  -普通用户的登录、注册
  -用户信息管理(关联的订单、商品信息)
  -商品信息管理(关联的卖家、订单信息，商品的发布、浏览)
  -订单信息管理(关联的商品、卖家、买家信息)

##系统要求
  -jdk17

##项目架构
  -技术栈
    .前端
    .后端：springboot、mybatis-plus、
    .数据库：mysql8.0
    .中间件：Redis缓存、RabbitMQ消息队列
    .消息推送
    .云存储
